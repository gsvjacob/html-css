Trabalhando com Listas

Listas ordenadas:

Para criar listas ordenadas utilizamos a tag "<ol>" e para cada item que desejarmos adicionar utilizamos a tag "<li>":

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar

  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar

Podemos alterar o estilo de numeração utilizando o parâmetro "type", além de mudar o valor inicial utilizando o parâmetro "start"

Listas não ordenadas

Para criar listas não ordenadas utilizamos a tag "<ul>" e para cada item que desejarmos adicionar utilizamos a tag "<li>":

  • Acordar
  • Tomar café
  • Escovar os dentes
  • Estudar
  • Almoçar
  • Acordar
  • Tomar café
  • Escovar os dentes
  • Estudar
  • Almoçar
  • Acordar
  • Tomar café
  • Escovar os dentes
  • Estudar
  • Almoçar

Podemos alterar o estilo de numeração utilizando o parâmetro "type"

Junção de listas:

Podemos colocar listas dentro de listas utilizando uma dentro da outra as tags citadas anteriormente, além de utilizar diferentes tipos de lista

  1. Acordar
    • Seg - Qua: 6:30
    • Qui - Sex: 7:30
    • Sab - Dom: 10:00
  2. Tomar café

Lista de definições

Para criar listas de definição utilizamos a tag "<dl>", adicionando itens com a tag "<dt>", e sua definição com a tag "<dd>"
