Controle de tamanho dos itens em flexbox:

Flex-grow

Define se um item pode (1) ou não (0) crescer de acordo com o contâiner


flex-grow: 0;

flex-grow: 1;

Flex-shrink

Define se um item pode (1) ou não (0) diminuir de acordo com o contâiner


flex-shrink: 1;

flex-shrink: 0;

Importante:

A W3C recomenda utilizar o shorthand "flex" no lugar de declarar individualmente as propriedades "grow" "shrink" "basis"

A ordem sempre deve ser: flex: {grow} {shrink} {basis};

Observações:

Caso você não declare o "flex" ele sera definido por padrão como "flex: 0 1 auto;", e caso você deseje declarar mesmo sendo as configurações padrões, você pode utilizar "flex: initial;"

Caso você deseje itens não tão flexíveis você pode utilziar "flex: none;" que define os valores para "flex: 0 0 auto;"

Caso você deseje itens mais flexíveis você pode utilizar "flex: auto;" que define os valores para "flex: 1 1 auto;"

Caso você declare somente dois ou menos valores, o restante receberá um valor padrão. Por exemplo "flex: 3" = "flex: 3 1 auto;"
