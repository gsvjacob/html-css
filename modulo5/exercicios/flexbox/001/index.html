flex-direction:

- row: É o padrão, define a direção horizontalmente da esquerda para a direita.

- row-reverse: Semelhante ao "row", porém da direita para a esquerda (OBS: se o navegador ou idioma tiver por padrão uma orientação assim, não é necessário adicionar o "reverse")

- collumn: define a direção verticalmente da cima para baixo

- collumn-reverse: Semelhante ao collumn, porém de baixo para cima (OBS: se o navegador ou idioma tiver por padrão uma orientação assim, não é necessário adicionar o "reverse")
