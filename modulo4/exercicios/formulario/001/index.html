Observações:

O parâmetro "label" deve ser usado para relacionar um texto (ex.: Nome) com sua entrada de dados para o formulário (ex.: Caixa de texto ao lado de "Nome")

O parâmetro "action" é usado para definir o local para onde os dados do formulário serão enviados

GET: Deixa visível os dados na URL do site após o envio do formulário

POST: Os dados ficam acessíveis, porém não explicitamente

- É preferível utilizar o método POST para o envio de dados sensíveis, apesar de não protegê-los devidamente (Para isso deve-se utilizar HTTPS)

- É preferível utilizar o método GET para dado não tão sensíveis, e para casos onde haja a possibilidade de compartilhar a URL, por exemplo em pesquisas do Google.

O método GET suporta no máximo 3000 bytes
