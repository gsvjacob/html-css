Extra:

select: Permite a criação de um campo onde o usuário pode escolher uma opção de acordo com as opções presentes. Para adiconar/criar opções basta utilizar a tag "<option>"

optgroup: Permite o agrupamento de opções dentro da área de escolha. As opções dentro de "optgroup" serão agrupadas em uma categoria no local de escolha

datalist: Permite adicionar opções/sugestões a um campo de texto

textarea: Permite criar uma área para o usuário enviar mensagens. É recomendável utilizar somente com o método POST, devido o limite de 300bytes do método GET
